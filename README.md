## 🚀 Quick Start

Clone it. Enter the lab. Push buttons.

### 1️⃣ Install dependencies

```powershell
npm install
```

Or if you're one of *those* people:

```powershell
pnpm install
# or
yarn
```

---

### 2️⃣ Run the dev server

```powershell
npm run dev
```

Then open whatever URL Vite prints in the terminal.

Usually something like:

```
http://localhost:5173
```
